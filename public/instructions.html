Schema of database for Cute Zoo

Three roles: logged in user(U), guest (G), zookeeper (Z)

Different functionalities depending on user status/role with resource-level permissions

  • Register, register as zookeeper, login, logout (G)
  • Logout (U, Z)
  • See a list of all records for each table (G, U)
  • See a list of all records zookeepers and food (Z)
  • See a list of supervised animal and its permitted meals (Z)
  • See detailed views for each table (U, Z)
  • Perform adding, updating, deleting records to the database (U)
  • Has the permission to modify and delete data only about supervised animals (Z)
  • Has the permission to add and delete permitted meals only for supervised animals (Z)
